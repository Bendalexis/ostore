com.persist

Bulk

object Bulk extends AnyRef

The Bulk loader moves multiple items in and out of OStore tables.

The external form used is a sequence of lines. Each line specifies a single item and consists of the item JSON key followed by a tab followed by the item JSON value.

This is the same format used by the UI Upload and Download operations.
